# jsonltool
Small JSONl Tool for ioBroker JSONL databases

## With npm >= 7

```bash
npm x --yes @iobroker/jsonltool@latest /opt/iobroker/iobroker-data
```

## With npm < 7

```bash
cd /opt/iobroker/iobroker-data
npx @iobroker/jsonltool@latest
```
